Un Aperçu du Contenu Détaillé

Introduction

Pourquoi apprendre HTML & CSS aujourd'hui ? Ce que tu vas accomplir avec ce guide. À qui s'adresse ce livre? Ce dont tu as besoin pour commencer.

Chapitre 1 - Découvrir le HTML: la structure d'une page web

Qu'est-ce que le HTML? Outils nécessaires (éditeur de code, navigateur). Structure d'un fichier HTML. Les balises essentielles. Écrire ta première page HTML. Mini exercice: créer une page "Hello World".

Chapitre 2 - Comprendre le CSS: la mise en forme

Qu'est-ce que le CSS? Comment relier HTML et CSS. Sélecteurs, classes, id. Propriétés courantes. Mise en page simple. Mini exercice: styliser ta page "Hello World".

Chapitre 3 - Créer une page responsive

Qu'est-ce que le responsive design? Meta viewport. Flexbox ou Grid. Media queries de base. Astuces pour tester. Mini exercice: transformer ta page en version responsive.

Chapitre 4 - Structurer un vrai projet HTML/CSS

Organisation des fichiers. Bonnes pratiques de nommage et de commentaires. Ajouter une navigation, une image de profil ou une bannière, une section "À propos". Mini projet guidé: construire un mini CV web/portfolio.

Chapitre 5 - Aller plus loin (bonus)

Où héberger ton site gratuitement (GitHub Pages, Netlify). Ressources utiles pour progresser (MDN, W3Schools, Frontend Mentor). Bases du SEO. Préparer son code pour un site professionnel.
